# Auto-Memory 插件

**版本**：v1.0.0（2026-02-25 创建）  
**作者**：CEO（修行者）  
**用途**：对话后自动写入 memory/，不需要手动记录

---

## 🎯 功能

### 自动记录
- 每次对话结束后
- 自动写入 `memory/YYYY-MM-DD.md`
- 包含：主题/决策/待办/未决

### 自动归档
- 检测重要对话（关键词：修行/源码/插件/重要）
- 自动归档到 `memory/important/`
- 方便后续检索

---

## 🔧 配置

```json
{
  "auto-memory": {
    "enabled": true,
    "memoryPath": "~/.openclaw/workspace/memory/",
    "importantKeywords": ["修行", "源码", "插件", "重要", "传道"],
    "autoArchive": true
  }
}
```
